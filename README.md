# AI Trading Bot

This is an automated trading application with Binance integration and AI-powered market analysis.

## Features
- **Binance Integration**: Securely connect your Binance account using API Keys.
- **Real-time Balance**: View your wallet balance directly on the dashboard.
- **AI Analysis**: Get market insights and trading signals (Buy/Sell/Hold) generated by the AI engine.
- **Modern UI**: A sleek, dark-themed interface designed for ease of use.

## Setup

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Application**:
   ```bash
   uvicorn app.main:app --reload
   ```

3. **Access the App**:
   Open your browser and navigate to `http://127.0.0.1:8000`.

## Usage

1. Enter your Binance API Key and Secret Key on the login page.
   - *Note: Keys are not stored permanently for security reasons.*
2. Once logged in, you will see your dashboard.
3. Use the "AI Auto-Trader" section to analyze trading pairs like BTC/USDT.
